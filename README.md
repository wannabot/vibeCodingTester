# Cat Variables Atelier

고양이 애호가 감성으로 꾸민 정적 웹페이지입니다.  
기존의 단순 랜덤 메시지 버튼 페이지를, 고양이 변수별 이미지 카드와 스포트라이트 섹션이 있는 갤러리형 랜딩 페이지로 리디자인했습니다.

## 변경 요약

### 1. 화면 구조 개편
- 단일 버튼 중심 UI를 고양이 테마 랜딩 페이지로 전면 변경
- 히어로 소개 영역, 오늘의 추천 변수, 변수 카드 갤러리 섹션 추가
- 모바일에서도 자연스럽게 보이도록 반응형 레이아웃 적용

### 2. 고양이 변수 기반 콘텐츠 추가
- `햇살 낮잠 지수`
- `호기심 레이더`
- `간식 감지력`
- `새벽 질주 모드`

각 변수마다 아래 정보를 함께 보여줍니다.
- 변수 이름
- 상태값
- 설명 문구
- GPT 이미지 생성용 프롬프트
- 해당 무드를 표현하는 이미지

### 3. 비주얼 스타일 리디자인
- 따뜻한 크림, 허니, 세이지 계열 컬러 사용
- 카드형 매거진 레이아웃 적용
- 고양이 애호가 느낌의 부드럽고 포근한 분위기로 재구성
- 버튼 클릭 시 추천 변수와 강조 카드가 함께 바뀌도록 인터랙션 추가

### 4. 이미지 구성 방식
- 현재 버전은 즉시 렌더링 가능하도록 인라인 SVG 고양이 일러스트를 사용
- 동시에 각 카드에 실제 GPT 이미지 생성에 활용할 수 있는 프롬프트를 포함
- 이후 OpenAI 이미지 API와 연결하면 실사풍 또는 일러스트풍 이미지로 확장 가능

## 파일 구성

- `index.html`: 메인 랜딩 페이지 구조
- `style.css`: 전체 레이아웃, 색상, 카드 스타일, 반응형 처리
- `script.js`: 고양이 변수 데이터, 카드 렌더링, 랜덤 스포트라이트 로직
- `upload/index.html`: 동일 화면을 참조하는 업로드용 진입 파일

## 확인 포인트

- 버튼을 누르면 오늘의 추천 고양이 변수가 랜덤으로 변경됩니다.
- 선택된 변수 카드는 강조 표시됩니다.
- 각 카드에서 GPT 이미지 생성용 프롬프트를 바로 확인할 수 있습니다.
